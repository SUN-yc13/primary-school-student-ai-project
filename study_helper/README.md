# 学习效率助手

一款本地化学习效率工具，面向学生提升时间管理与专注能力。

## 功能特性

### 1. 番茄专注计时
- 自定义专注时长（15/25/30/45/50/60分钟快捷设置）
- 自动短休息/长休息，支持自定义时长
- 弹窗提醒，番茄完成通知
- 进度条实时显示
- 今日番茄数统计

### 2. 本地待办清单
- 添加、删除、完成待办事项
- 三级优先级（高/中/低）
- 双击切换完成状态
- 右键菜单快捷操作
- 一键清除已完成

### 3. 每日学习时长统计
- 今日专注时长和番茄数
- 累计总时长和总番茄数
- 本周每日统计详情
- 数据自动本地保存

### 4. 置顶便签
- 桌面便签始终置顶显示
- 拖拽移动，自由调整大小
- 6种颜色可选（黄/绿/蓝/粉/橙/紫）
- 内容自动保存
- 支持显示/隐藏全部便签

### 5. 专注时段软件屏蔽
- 自定义屏蔽应用列表
- 一键添加常见分心应用（微信、QQ、抖音等）
- 专注时自动屏蔽，暂停时自动恢复
- 可单独启用/禁用每个屏蔽项

## 技术特点

- **纯本地运行**：无账号、无联网要求，保护隐私
- **数据本地存储**：所有数据保存在用户目录 `.study_helper` 文件夹
- **Windows 原生风格**：基于 tkinter，适配系统主题
- **多标签页界面**：功能分区清晰，操作直观
- **轻量高效**：启动快，占用资源少

## 使用方法

### 直接运行（需 Python 环境）

```bash
# 安装依赖
pip install psutil

# 运行程序
python main.py
```

### 打包为 exe

```bash
# 运行打包脚本
build.bat
```

打包完成后，可执行文件位于 `dist/学习效率助手.exe`

### 制作安装包

1. 先运行 `build.bat` 打包 exe
2. 安装 [Inno Setup](https://jrsoftware.org/isinfo.php)
3. 使用 Inno Setup 编译器打开 `installer.iss`
4. 编译生成安装包

## 数据存储

所有用户数据保存在：
```
C:\Users\用户名\.study_helper\
```

包含以下文件：
- `todos.json` - 待办清单
- `statistics.json` - 学习统计
- `settings.json` - 设置配置
- `notes.json` - 便签数据
- `blocklist.json` - 屏蔽应用列表

可在程序的「设置」页面中直接打开数据文件夹。

## 目录结构

```
study_helper/
├── main.py          # 主程序入口（GUI界面）
├── data_manager.py  # 数据管理模块
├── pomodoro.py      # 番茄钟计时模块
├── todo.py          # 待办清单模块（已整合到main）
├── app_blocker.py   # 应用屏蔽模块
├── sticky_note.py   # 置顶便签模块
├── build.bat        # 打包脚本
├── installer.iss    # Inno Setup 安装脚本
├── requirements.txt # 依赖列表
└── README.md        # 使用说明
```

## 快捷键

- **待办清单**：双击切换完成状态，Delete 键删除选中项
- **应用屏蔽**：双击切换启用/禁用状态

## 注意事项

- 应用屏蔽功能需要管理员权限才能关闭某些系统级进程
- 便签窗口无边框，可通过顶部拖拽移动
- 番茄钟数据每日自动统计，不可手动修改
- 卸载程序不会自动删除用户数据，如需清除请手动删除 `.study_helper` 文件夹
