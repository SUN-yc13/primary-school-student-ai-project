# 桌面文件智能管理助手

一款 Windows 本地文件整理工具，解决桌面杂乱、文件管理低效的痛点。

## 功能特性

### 1. 文件整理
- 一键按类型整理桌面/下载/任意文件夹
- 自动分类：图片、文档、视频、音频、压缩包、程序、其他
- 支持预览模式（不实际移动文件）
- 自动处理重名文件

### 2. 批量重命名
- 多种重命名规则：添加前缀、添加后缀、序号命名、文本替换、修改扩展名
- 支持正则表达式替换
- 预览功能，确认后再执行
- 可选择是否包含子文件夹

### 3. 重复文件检测
- 基于 MD5 哈希值精确识别重复文件
- 先按文件大小筛选，再计算哈希，效率更高
- 可设置最小文件大小，忽略小文件
- 分组显示重复文件，支持选择删除
- 支持移动到回收站（需安装 send2trash）

### 4. 大文件扫描
- 快速扫描定位大文件
- 可自定义最小文件大小（MB）
- 按大小降序排列，一目了然
- 支持打开文件所在位置
- 可扫描文件夹大小

## 技术特点

- **纯本地运行**：无需联网，保护隐私
- **零额外依赖**：核心功能全部使用 Python 标准库
- **Windows 原生风格**：基于 tkinter，适配系统主题
- **多线程处理**：耗时操作后台运行，界面不卡顿

## 使用方法

### 直接运行（需 Python 环境）

```bash
# 安装可选依赖（推荐）
pip install send2trash

# 运行程序
python main.py
```

### 打包为 exe

```bash
# 运行打包脚本
build.bat
```

打包完成后，可执行文件位于 `dist/桌面文件智能管理助手.exe`

### 制作安装包

1. 先运行 `build.bat` 打包 exe
2. 安装 [Inno Setup](https://jrsoftware.org/isinfo.php)
3. 使用 Inno Setup 编译器打开 `installer.iss`
4. 编译生成安装包

## 目录结构

```
file_manager/
├── main.py          # 主程序入口（GUI界面）
├── organizer.py     # 文件整理模块
├── renamer.py       # 批量重命名模块
├── duplicate.py     # 重复文件检测模块
├── largefile.py     # 大文件扫描模块
├── build.bat        # 打包脚本
├── installer.iss    # Inno Setup 安装脚本
├── requirements.txt # 依赖列表
└── README.md        # 使用说明
```

## 注意事项

- 整理文件前建议先使用预览模式确认效果
- 删除文件时默认移动到回收站（需安装 send2trash）
- 大文件扫描系统盘可能需要较长时间
- 包含子文件夹时请谨慎操作，避免误移动系统文件
